# 🎯 How to Start - Choose Your Path

## You Have 2 Options:

---

## 🟢 Option 1: Full Setup (Recommended) - 5 Minutes

**Best for:** Going live immediately with your own database

### What You'll Get:
✅ Website live on internet  
✅ Your own database  
✅ Can add unlimited videos  
✅ Admin panel fully functional  
✅ Contact forms save to database  

### Follow This Guide:
📖 **Open:** `/START_HERE.md`

### Quick Steps:
1. `npm install`
2. Create Supabase account (free)
3. Add keys to `.env`
4. Run database migration
5. Create admin user
6. `npm start` to test
7. Deploy to Vercel

**Time:** 5 minutes  
**Result:** Fully working website online

---

## 🟡 Option 2: Quick Test (Development) - 1 Minute

**Best for:** Just want to see it work locally first

### What You'll Get:
✅ Website works on your computer  
✅ Gallery shows 3 sample videos  
✅ Can test all features  
⚠️ Uses mock data (not saved)  
⚠️ Only you can see it  

### Quick Steps:
1. `npm install`
2. `npm start`
3. Open browser

**Time:** 1 minute  
**Result:** Working demo on localhost
